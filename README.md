# 💰 配当管理（haitou-app）

株の配当金を記録・集計するスマホ向けPWA。インストール不要、**データはすべて自分の端末内（localStorage）に保存**され、サーバーには一切送信されません。

## 機能
- **保有銘柄**：株数・1株配当を入力 → 年間配当見込み・利回りを自動計算
- **NISA / 課税口座の区分**：NISAは非課税、課税口座は源泉20.315%を引いた「税引後の手取り見込み」を表示
- **受取記録**：振り込まれた配当を日付つきで記録（月別グラフ・年間実績に反映）
- **バックアップ**：データをJSONで書き出し／復元
- **オフライン対応**：一度開けば圏外でも使える（Service Worker）

## 使い方（iPhoneのホーム画面に追加）
1. 公開URLをSafariで開く
2. 共有ボタン → 「ホーム画面に追加」
3. アプリのように起動できます

## プライバシー
- 公開されるのはプログラム（HTML/JS）のみ。保有銘柄・株数・金額などの**個人データは含まれません**。
- バックアップJSONには実データが入るため、`.gitignore` で公開対象から除外しています。

## ローカルで動かす
```
python -m http.server 8731
```
ブラウザで `http://localhost:8731` を開く。
